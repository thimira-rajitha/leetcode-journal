## About

This repository is my personal LeetCode journal for tracking problem-solving progress, practicing multiple languages, and keeping short notes for revision.

Each problem lives in its own folder and may include:

- a problem-specific `README.md`
- one or more language implementations
- approach notes
- time and space complexity

## Notes

- One folder is used per problem
- A problem may belong to multiple tags
- Notes and explanations are written inside each problem's `README.md`
- Not every problem needs all four language implementations from the start

## Workflow

For each new problem:

1. Create a new folder inside `problems/`
2. Add a `README.md`
3. Add one or more solution files
4. Update the Problem Index in this file
5. Update the progress section
